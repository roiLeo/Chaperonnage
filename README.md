# Chaperonnage
============

A Symfony project created on April 11, 2017, 3:32 pm.
php 7.1 

## Installation
Clone repos
```
git-clone
```
Install the package
```
composer install
```
Install assets
```
assets install
```
Database creation
```
doctrine:database:create
```
Generate a database migration
```
doctrine:migrations:migrate (diff)
```
Generate Data 
```
doctrine:fixtures:load
```
